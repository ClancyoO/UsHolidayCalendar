# 节日日历应用

这是一个交互式节日日历应用，允许用户查看不同年份的节日信息，并获取节日的中英文详情。应用采用现代化的液态玻璃设计风格，提供直观、美观的用户界面。

## 功能特点

1. **年份选择器**：允许用户选择不同年份查看对应的节日信息
2. **交互式日历视图**：以月历形式展示全年节日分布，节日显示为中文翻译
3. **节日详情展示**：点击节日后显示英文名称和中文介绍，以及提供的数据内的对应活动区域
4. **智能搜索功能**：用户可通过关键词快速查找特定节日
5. **液态玻璃设计**：采用现代化的UI设计，提供流畅的视觉体验

## 技术实现

- **前端框架**：HTML5, CSS3, JavaScript (ES6+)
- **样式库**：Tailwind CSS v3
- **图标库**：Font Awesome
- **数据存储**：JavaScript对象存储节日数据

## 如何使用

1. 打开 `index.html` 文件即可运行应用
2. 使用顶部的年份选择器切换不同年份
3. 点击月份导航按钮查看不同月份的节日
4. 点击日历中的节日名称查看详细信息
5. 使用搜索框快速查找特定节日

## 数据说明

节日数据存储在 `holidays.js` 文件中，以JavaScript对象的形式组织。每个节日包含以下信息：
- `date`：日期（YYYY-MM-DD格式）
- `name`：英文名称
- `type`：节日类型（如Federal Holiday, State Holiday等）
- `details`：适用地区详情
- `chineseName`：中文名称
- `description`：中文介绍

要添加新的年份数据，只需在 `holidays.js` 文件中按照现有格式添加新的年份数据对象，并在 `holidays` 对象中添加对应的引用。
